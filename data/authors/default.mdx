# 关于我们

『多元寻迹』创建于 **2024 年**，是一个聚焦跨性别、变装（CDTS）、性转、伪娘等多元性别表达与文化的专业搜索与内容导航平台。我们致力于通过先进的数据检索技术与AI辅助导航，帮助用户快速准确地找到高质量且真实可信的相关资源。同时，通过长期存档与索引，『多元寻迹』保护那些易被忽略的多元文化表达与故事，让每个独特的灵魂都被温柔地看见与记录。

作为专业搜索平台，我们关注跨性别、非二元性别、间性人，以及其他打破传统二元性别认知的多元群体，建立了涵盖数十万条相关数据的专业索引。在主流搜索引擎无法完全满足需求的环境中，『多元寻迹』为多元性别社群提供精准、全面的信息检索服务。

## 我们的使命

- **发现与连接**：为所有与多元性别相关的中文资源提供一个高效、精准的检索入口，包括个人故事、医疗资源、法律政策、文学艺术等多种形式。  
- **索引与可见性**：通过专业的搜索算法与分类体系，提高有价值内容的可见性，帮助用户快速定位所需信息，避免在信息海洋中迷失。  
- **多样性与包容**：秉承多元包容的理念，确保从 mtf/ftm 到非二元、CDTS、间性人等各种身份与表达的相关内容都能被公平索引，让性与性别的多元性在搜索结果中得到充分体现。

## 创建背景
在数字时代，尽管信息爆炸式增长，但多元性别相关的珍贵资源却往往难以通过主流搜索引擎有效获取。传统搜索引擎受限于通用算法和商业导向，对多元性别议题的内容索引往往不够精准或全面，甚至因内容审查而过滤掉重要信息。随着中文互联网生态的变迁，许多跨性别者、非二元群体的声音也在搜索结果中逐渐被边缘化。我们相信，**专业搜索胜于盲目筛选**——唯有构建针对性的搜索引擎，才能真正连接用户与他们所需的多元性别资源。

## 主要功能与特色

1. **专业内容索引**  
   『多元寻迹』专门爬取并索引跨性别医疗信息、社会新闻、法律政策解读、学术研究文献、变身文学与小说、性转与伪娘漫画等多元性别相关内容。通过深度语义分析和专业分类标签，我们能够提供比通用搜索引擎更加精准的检索体验。  
   同时，为响应不同用户的搜索需求，我们设计了多维度的筛选功能和主题目录导航，帮助用户更高效地发现并获取所需资源。

2. **智能内容分类**  

   我们理解多元性别资源的复杂性和多样性，因此开发了智能分类系统，能够根据内容特征和受众适宜度进行分级展示。对于含有成人元素的内容，如变身文学的限制级篇章、扶她与双性题材、Eunuch 相关资源等，我们在"成人内容"专区中单独索引，并设置了相应的访问提示。
   这一设计既尊重了不同用户的信息需求，也确保一般检索时不会意外展示不适宜的内容。我们严格过滤任何涉及违法内容的索引，保持搜索环境的健康与合规。相关成人内容索引服务请访问 [cdtsf.com](https://cdtsf.com)。

3. **高级搜索与智能推荐**  
   我们的搜索引擎不仅提供基础的关键词检索，还支持高级语义搜索、多条件组合查询、历史检索记录分析等功能。通过机器学习技术，系统能够理解用户的搜索意图，并在结果中智能排序最相关和高质量的内容。
   
   此外，我们还开发了个性化推荐功能，根据用户的搜索历史和兴趣偏好，主动推荐可能有价值的相关资源，帮助用户发现更多未知但有益的内容。

4. **开放协作与社区改进**  
   搜索引擎的优化离不开用户的反馈与参与。任何人都可以通过 GitHub 或网页表单提交新的网站、资源或未被索引的内容建议。我们鼓励社区成员帮助识别高质量资源，报告无效链接，并参与搜索算法和分类体系的完善。  
   通过持续改进和社区协作，我们致力于打造一个越来越精准、全面的多元性别专业搜索平台。

5. **知识图谱与资源导航**  
   除了传统的搜索功能，『多元寻迹』还构建了多元性别领域的知识图谱，揭示不同概念、资源和信息之间的关联性。用户可以通过这一可视化导航系统，探索从一个起点延伸出的相关主题和资源网络，获得更全面、立体的认知体验。
   
   这一功能特别适合初入多元性别领域的探索者，以及希望进行深入研究的学者和支持者。

## 内容范围与原则

- **索引范围**：专注于所有与多元性别相关或不符传统二元性别划分的中文资源，包括但不限于个人博客、社交媒体账号、专业网站、学术论文、新闻报道、艺术创作等。  
- **多元视角**：我们索引各种立场的内容，既包括正面科普与学术探讨，也包括具有争议性的观点，以提供全面的信息视角，供用户自行判断与分析。  
- **质量与可靠性**：我们通过算法和人工审核相结合的方式，努力提高搜索结果的质量和可靠性，但不对任何被索引内容的准确性或时效性做绝对保证。请用户在使用信息时保持批判思考。  
- **内容分级**：对于成人或限制级内容，我们实施严格的分级索引策略，确保这类内容不会在一般搜索中无预警展示。同时，我们绝不索引任何违法内容。  

## 法律与伦理声明

- **链接与索引**：作为搜索引擎，我们主要提供内容的链接和索引，而非直接存储或复制原始内容。我们尊重原作者的知识产权，如有侵权疑虑，请通过正式渠道联系我们处理。  
- **隐私保护**：我们重视用户的搜索隐私，不会记录可识别个人身份的搜索历史，也不会将用户数据用于商业目的或分享给第三方。  
- **非商用原则**：『多元寻迹』坚持非营利、非商业化运营，所有功能对用户完全免费，不植入广告或收集用户数据进行商业变现。

## 技术架构与运营

- **项目性质**：『多元寻迹』由技术志愿者与AI专家联合开发维护，基于开源技术构建，秉持开放、中立、服务社群的理念。  
- **搜索技术**：我们采用最新的自然语言处理技术和知识图谱算法，针对多元性别领域特殊术语和语境进行了专门优化，提供更精准的搜索体验。  
- **持续更新**：我们的爬虫系统定期更新索引库，及时收录新内容并清理失效链接，确保搜索结果的时效性和可用性。
